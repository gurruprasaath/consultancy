# Food7 - AI Restaurant Management System

## Quick Start

### 1. Backend Setup

```bash
cd backend
npm install
cp .env.example .env
# Add your API keys to .env
npm run dev
```

### 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### 3. Login

- URL: http://localhost:5173
- Admin: admin@food7.com / admin123

## API Keys Required

- OPENAI_API_KEY (for Whisper)
- GROQ_API_KEY (for LLM)
